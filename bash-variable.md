# Bash 變數

此區包含系統預設的變數及自訂變數的使用方式

### 系統預設變數
環境變數 | 說明
--------|:----
HOME    |當前使用者的家目錄
PATH    |搜尋命令的路徑
PS1     |命令提示符號，通常由 `\u@\h:\w\$` 組合，見[如下說明](#shellscript-can-shu)
PS2     |第二層的命令提示符，用在額外的輸入模式，一般為 `>`
IFS     |輸入欄位的區隔符號
$0      |當前執行shell script的名稱
$#      |傳入參數的數量
$$      |shell script 的 PID

### ShellScript參數
參數      | 說明
----------|:----
$1,$2,... |提取輸入的參數
$*        |所有參數的串列，以IFS的字元做區隔
$@        |所有參數的串列，不使用IFS並且展開所有傳入的變數
